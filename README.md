## Run the app
- `npm run dev`

### Back-End
- `npm i express bcryptjs jsonwebtoken config express-validator mongoose`
- `npm i -D nodemon concurrently`

### Front-End
- `npx yarn create react-app client`
- `npm i axios react-router-dom uuid react-transition-group`
